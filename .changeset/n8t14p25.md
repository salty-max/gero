**`gero disasm --check-roundtrip` ‚Äî CLI round-trip gate**

New flag that drives a `.gx` archive through `asm ‚Üí disasm ‚Üí asm` via
the existing `disasm.roundTripArchive` helper and exits non-zero if
the re-assembled base image doesn't match the original byte-for-byte.
`--quiet` suppresses the one-line ok summary.

Wired into `scripts/test-examples.sh` so every shipped example
(`hello.gas`, `fib.gas`, `counter.gas`, `save.gas`,
`banks/main.gas`) is gated end-to-end at the CLI boundary on every
PR ‚Äî the existing library-level round-trip tests in
`tests/disasm/roundtrip.test.zig` keep validating the helper itself.

Bank bytes are excluded from the compare: `roundTripArchive` only
disassembles the base image today, so a multi-bank archive's
`header.banks` always disagrees with the re-assembled side. Extending
the helper to cover bank slots is tracked separately.
