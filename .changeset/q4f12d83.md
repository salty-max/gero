Add the six `.gas` operator/punctuation tokens that the v0.1 spec
uses but the initial lexer omitted: `-` (`minus`), `*` (`star`),
`<` (`lt`), `>` (`gt`), `.` (`dot`), and a bare `&` (`ampersand`)
for `&[ ]` address-expression openers and `&r1` register
pointers. The bare-`&` parser refuses when followed by a hex
digit so the structured `addrLiteral` error path still wins
for malformed cases like `&12345`.
