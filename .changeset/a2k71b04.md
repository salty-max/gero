Finalize the `feat(disasm): round-trip integration + --bank filter` —
closes #41.

**`gero disasm` whole-cart default mode**
When `--bank` is omitted, the CLI now renders the base image followed
by every bank, each prefixed with a `; --- base image ---` /
`; --- bank N ---` section header. Single-cart programs (no banks)
render unchanged. `--bank=N` still scopes the view to a single bank.

**Round-trip CI gate for example programs**
New `disasm.roundTripArchive` helper drives a full `.gx` archive
through asm → disasm → asm and returns the re-assembled archive.
Unlike the older `roundTripImage`, it reads the archive's debug
section and replays data blocks as `data8 NAME = ...` directives so
sources mixing code and data survive the trip.

`tests/disasm/roundtrip.test.zig` now exercises `hello.gas`, `fib.gas`,
and `counter.gas` end-to-end on every PR — any change that breaks the
disasm's losslessness fails CI.

**Printer API**
`PrintOptions` gains `show_addresses: bool` (default `true`) so
callers can keep symbol-address mapping without the `XXXX:` gutter —
the round-trip path needs the former without the latter.
